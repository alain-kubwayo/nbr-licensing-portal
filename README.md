# NBR Licensing Portal

This is BNR's Bank Licensing & Compliance Portal built with:

- **Backend**: NestJS API + PostgreSQL + Swagger docs
- **Frontend**: React + TypeScript + Vite web UI

## Prerequisites

- Node.js 20+
- pnpm
- Docker (for the database)
